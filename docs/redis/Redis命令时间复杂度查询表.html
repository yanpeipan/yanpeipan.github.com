命令时间复杂度

Zset有序集合​

命令时间复杂度
zaddO(k*log(n))，k为添加 成员个数，n为当前成员个数
zcardO(1)
zscoreO(1)
zrank

zrevrank		O(log(n))，n为当前成员个数

zrange

zrevrange		O(log(n)+k)，k为要获取成员个数，n为当前成员个数
zrangebyscore

zrevrangebyscore		O(log(n)+k)，k为要获取成员个数，n为当前成员个数

zremrangebyrankO(log(n)+k)，k为要删除成员个数，n为当前成员个数
